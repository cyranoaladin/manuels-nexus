# Perimetre Terminale specialite mathematiques (TSPE)

## Source reglementaire

Programme de specialite mathematiques, classe terminale generale.
BO applicable : **a confirmer** (BO 2020 ou revision 2026).

> **R7** : le texte officiel n'est pas dans le depot. Les 13 chapitres ci-dessous
> sont une estimation basee sur la structure historique du programme. La liste
> definitive et les capacites par chapitre sont **A_VALIDER_HUMAIN** contre le BO
> en vigueur.

## Architecture provisoire (13 chapitres)

| # | Chapitre | Theme BO |
|---|---|---|
| 1 | TSPE-SUITES-LIMITES | Suites : limites, theoremes de convergence |
| 2 | TSPE-FONCTIONS-LIMITES | Fonctions : limites, asymptotes |
| 3 | TSPE-CONTINUITE | Continuite, TVI |
| 4 | TSPE-DERIVATION | Derivation : complementes (convexite, points d'inflexion) |
| 5 | TSPE-LOGARITHME | Fonction logarithme neperien |
| 6 | TSPE-PRIMITIVES-INTEGRALES | Primitives et integrales |
| 7 | TSPE-EQUATIONS-DIFFERENTIELLES | Equations differentielles y'=ay+b |
| 8 | TSPE-COMBINATOIRE-DENOMBREMENT | Combinatoire et denombrement |
| 9 | TSPE-PROBABILITES | Probabilites : lois a densite, loi normale |
| 10 | TSPE-GEOMETRIE-ESPACE | Geometrie dans l'espace (orthogonalite, distances) |
| 11 | TSPE-NOMBRES-COMPLEXES | Nombres complexes |
| 12 | TSPE-ARITHMETIQUE | Arithmetique (divisibilite, congruences) |
| 13 | TSPE-MATRICES | Matrices et graphes (si au programme) |

> Cette liste est provisoire. Les chapitres 12 et 13 dependent du programme
> effectivement en vigueur. Certains chapitres peuvent etre fusionnes ou
> scindes selon le decoupage officiel.

## Workflow

Meme pipeline que 1SPE :
- LOT 0 : contrat depuis referentiel BO
- LOT 1-2 : curation
- LOT 3 : cours + methodes
- LOT 4 : 50 exercices + corriges
- LOT 5 : QCM + remediation
- LOT 6 : evaluations A+B + resolution aveugle
- LOT 7 : assemblage + PDF + tag

## Prerequis pour demarrer

1. Fournir le texte BO Terminale dans `sources/bo_tspe_maths.pdf`
2. Valider la liste des 13 chapitres
3. Creer les `capacites_TSPE_*.json` depuis le BO
