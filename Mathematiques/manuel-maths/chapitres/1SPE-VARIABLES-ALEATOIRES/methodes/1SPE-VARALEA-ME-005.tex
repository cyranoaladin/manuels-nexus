\begin{fichemethode}{M5}{Résoudre un problème contextualisé avec une variable aléatoire}

\quandUtiliser{%
  Utiliser cette méthode pour les problèmes de jeux de hasard, d'assurance, de contrôle qualité ou de décision, où il faut modéliser une situation par une variable aléatoire et conclure grâce à l'espérance.
}

\pasApas{%
  \begin{enumerate}
    \item Je lis l'énoncé et j'identifie l'expérience aléatoire.
    \item Je définis clairement la variable aléatoire $X$ (par exemple : $X$ = gain algébrique du joueur).
    \item Je détermine toutes les valeurs possibles de $X$.
    \item Je calcule la probabilité de chaque valeur (arbre, tableau, dénombrement).
    \item Je construis le tableau de loi et je vérifie $\sum p_i = 1$.
    \item Je calcule $E(X)$ et j'interprète dans le contexte :
      \begin{itemize}
        \item Jeu : $E(X) > 0$ favorable, $E(X) = 0$ équitable, $E(X) < 0$ défavorable.
        \item Assurance : $E(B) > 0$ rentable pour l'assureur.
        \item Qualité : $E(X)$ donne le nombre moyen de défauts.
      \end{itemize}
  \end{enumerate}
}

\exempleRedige{%
  Un joueur paie $2$~euros pour lancer un dé. S'il obtient $6$, il reçoit $10$~euros. Sinon, il ne reçoit rien. Le jeu est-il favorable ?

  \medskip
  \commentaireMarge{Définir $X$.}%
  Soit $X$ le gain algébrique du joueur.

  \commentaireMarge{Valeurs et probabilités.}%
  Si $6$ : $X = 10 - 2 = 8$ avec $P = \frac{1}{6}$.
  Sinon : $X = 0 - 2 = -2$ avec $P = \frac{5}{6}$.

  \commentaireMarge{Espérance.}%
  $E(X) = 8 \times \frac{1}{6} + (-2) \times \frac{5}{6} = \frac{8 - 10}{6} = \frac{-2}{6} = -\frac{1}{3} \approx -0{,}33$.

  \commentaireMarge{Conclusion.}%
  $E(X) < 0$ : le jeu est défavorable au joueur. En moyenne, il perd environ $0{,}33$~euro par partie.
}

\pieges{%
  \begin{itemize}
    \item \textbf{Piège 1.} Oublier de soustraire la mise dans le gain algébrique.
    \item \textbf{Piège 2.} Ne pas définir $X$ explicitement au début de la résolution.
    \item \textbf{Piège 3.} Conclure sur le résultat d'une seule partie au lieu de raisonner « en moyenne ».
  \end{itemize}
}

\verifier{%
  Vérifier la cohérence de $E(X)$ avec le contexte : si la mise est élevée et les gains rares, $E(X)$ doit être négatif (jeu défavorable au joueur).
}
