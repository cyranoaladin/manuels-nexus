\[

\textbf{2.} Signe de $B'(x) = -3(x-1)(x-5)$ sur $[0\,;\,7]$ :
\begin{itemize}
\item $0 \leqslant x < 1$ : $(x-1) < 0$, $(x-5) < 0$, donc $B'(x) < 0$ (décroissant).
\item $1 < x < 5$ : $(x-1) > 0$, $(x-5) < 0$, donc $B'(x) > 0$ (croissant).
\item $5 < x \leqslant 7$ : $(x-1) > 0$, $(x-5) > 0$, donc $B'(x) < 0$ (décroissant).
\end{itemize}
