\textbf{Question 1.} On pose $u = x^2-3$ et $v = x^2+1$, donc $u' = 2x$ et $v' = 2x$.
\[
f'(x) = 2x(x^2+1) + (x^2-3)(2x) = 2x\bigl[(x^2+1) + (x^2-3)\bigr] = 2x(2x^2-2) = 4x(x^2-1).
\]
On factorise : $f'(x) = 4x(x-1)(x+1)$.

\textbf{Question 2.} $f'(x) = 0 \iff 4x(x-1)(x+1) = 0 \iff x \in \{-1\,;\,0\,;\,1\}$.
