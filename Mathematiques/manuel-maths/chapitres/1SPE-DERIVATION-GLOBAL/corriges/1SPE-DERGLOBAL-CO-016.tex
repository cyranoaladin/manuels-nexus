\textbf{Question 1.} On pose $u = x^2-2x$, $v = x^2+1$, donc $u' = 2x-2$, $v' = 2x$.
\[
f'(x) = \frac{(2x-2)(x^2+1) - (x^2-2x)(2x)}{(x^2+1)^2}.
\]
Développons le numérateur :
\[
(2x-2)(x^2+1) = 2x^3+2x-2x^2-2, \qquad (x^2-2x)(2x) = 2x^3-4x^2.
\]
\[
\text{Numérateur} = 2x^3+2x-2x^2-2 - 2x^3+4x^2 = 2x^2+2x-2 = 2(x^2+x-1).
\]
Donc $\displaystyle f'(x) = \frac{2(x^2+x-1)}{(x^2+1)^2}$.

\textbf{Question 2.} Puisque $(x^2+1)^2 > 0$, le signe de $f'(x)$ est celui de $x^2+x-1$. On résout $x^2+x-1=0$ :
\[
\Delta = 1 + 4 = 5, \qquad x = \frac{-1 \pm \sqrt{5}}{2}.
\]
On note $x_1 = \dfrac{-1-\sqrt{5}}{2} \approx -1{,}618$ et $x_2 = \dfrac{-1+\sqrt{5}}{2} \approx 0{,}618$.
