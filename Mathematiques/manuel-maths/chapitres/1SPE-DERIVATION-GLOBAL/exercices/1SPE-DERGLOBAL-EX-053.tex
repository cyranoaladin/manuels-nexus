\begin{exercice}{1SPE-DERGLOBAL-EX-053}{1}{6}
En utilisant la formule $(x^n)' = nx^{n-1}$ valable pour $n \in \mathbb{Z}$, dériver les fonctions suivantes sur $\mathbb{R} \setminus \{0\}$ :
\begin{enumerate}
  \item $f(x) = x^{-2}$
  \item $g(x) = x^{-3}$
  \item $h(x) = 5x^{-1}$
  \item $k(x) = x^{-4} + x^2$
\end{enumerate}
\end{exercice}
