\section*{TD — Optimisation du profit d'une entreprise}

\textit{Durée : 50 minutes.}

\bigskip

\begin{center}
\textbf{Situation}

\medskip
\parbox{0.9\textwidth}{Une petite entreprise fabrique et vend des stylos personnalisés. Le responsable dispose des informations suivantes :
\begin{itemize}
  \item le coût total de production de $x$ stylos (en centaines) est $C(x) = x^3 - 9x^2 + 30x + 5$ (en centaines d'euros) ;
  \item le prix de vente unitaire est $p(x) = 45 - 6x$ euros par cent stylos (avec $0 < x \leqslant 6$).
\end{itemize}
L'entreprise cherche à déterminer la quantité $x$ qui maximise son profit.}
\end{center}

\bigskip
