\commentaireMarge{Tangence $\iff$ $d(\Omega, \mathcal{D}) = r$.}

$d = \dfrac{|2 - 2(-1) + c|}{\sqrt{1 + 4}} = \dfrac{|4 + c|}{\sqrt{5}}$.

$d = \sqrt{5} \iff |4 + c| = 5$.

$4 + c = 5$ ou $4 + c = -5$, soit $\boxed{c = 1}$ ou $\boxed{c = -9}$.
