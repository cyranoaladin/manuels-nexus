Soustraction : $-8x - 6y + 46 = 0$, soit $4x + 3y - 23 = 0$, d'où $y = \dfrac{23 - 4x}{3}$.

Substitution dans $\mathcal{C}_1$ : $x^2 + \dfrac{(23-4x)^2}{9} = 25$.

$9x^2 + 529 - 184x + 16x^2 = 225$, soit $25x^2 - 184x + 304 = 0$.

$\Delta = 184^2 - 4 \times 25 \times 304 = 33856 - 30400 = 3456 = 576 \times 6$.

$\sqrt{\Delta} = 24\sqrt{6}$.

$x = \dfrac{184 \pm 24\sqrt{6}}{50} = \dfrac{92 \pm 12\sqrt{6}}{25}$.

Les ordonnées se déduisent de $y = \dfrac{23 - 4x}{3}$.
