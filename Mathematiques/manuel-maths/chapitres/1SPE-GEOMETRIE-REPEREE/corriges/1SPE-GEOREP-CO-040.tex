\commentaireMarge{Une droite passant par $O$ a pour équation $y = mx$, soit $mx - y = 0$.}

$d(\Omega, \mathcal{D}) = \dfrac{|3m - 4|}{\sqrt{m^2 + 1}} = 1$.

$(3m-4)^2 = m^2 + 1$, soit $9m^2 - 24m + 16 = m^2 + 1$, d'où $8m^2 - 24m + 15 = 0$.

$\Delta = 576 - 480 = 96$, $\sqrt{\Delta} = 4\sqrt{6}$.

$m = \dfrac{24 \pm 4\sqrt{6}}{16} = \dfrac{6 \pm \sqrt{6}}{4}$.

Les deux tangentes ont pour équations $y = \dfrac{6 + \sqrt{6}}{4}x$ et $y = \dfrac{6 - \sqrt{6}}{4}x$.
