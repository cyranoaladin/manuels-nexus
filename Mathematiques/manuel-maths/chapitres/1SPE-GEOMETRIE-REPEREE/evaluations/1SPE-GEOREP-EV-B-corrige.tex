\textbf{Exercice 1 — Équation cartésienne (4 pts)}

\textbf{1.} $\vec{AB}\begin{pmatrix} 6-2 \\ -1-5 \end{pmatrix} = \begin{pmatrix} 4 \\ -6 \end{pmatrix}$.

\textbf{2.} $m = \dfrac{-6}{4} = -\dfrac{3}{2}$.

\textbf{3.} $y - 5 = -\dfrac{3}{2}(x - 2)$, soit $2y - 10 = -3x + 6$, d'où $3x + 2y - 16 = 0$.

\textbf{4.} $A$ : $6 + 10 - 16 = 0$ \checkmark. $B$ : $18 - 2 - 16 = 0$ \checkmark.

\bigskip

\textbf{Exercice 2 — Vecteur normal (4 pts)}

\textbf{1.} $\vec{n}(3 ; 2)$ normal, $\vec{u}(-2 ; 3)$ directeur. $\vec{n} \cdot \vec{u} = -6 + 6 = 0$ \checkmark.

\textbf{2.} $\mathcal{D}'$ : vecteur normal $(-2 ; 3)$ (directeur de $\mathcal{D}$), passant par $M(1 ; 2)$ :

$-2(x-1) + 3(y-2) = 0$, soit $2x - 3y + 4 = 0$.

$M$ : $2 - 6 + 4 = 0$ \checkmark.

\textbf{3.} $3 \times 2 + 2 \times (-3) = 6 - 6 = 0$ \checkmark.

\bigskip

\textbf{Exercice 3 — Cercle (4 pts)}

\textbf{1.} $(x - 4)^2 + (y - 1)^2 = 9$.

\textbf{2.} $(7-4)^2 + (1-1)^2 = 9 + 0 = 9$. $P \in \mathcal{C}$.

\textbf{3.} $(4-4)^2 + (4-1)^2 = 0 + 9 = 9$. $Q \in \mathcal{C}$.

\textbf{4.} $x^2 - 8x + 16 + y^2 - 2y + 1 = 9$, soit $x^2 + y^2 - 8x - 2y + 8 = 0$.

Retour : $(x-4)^2 - 16 + (y-1)^2 - 1 + 8 = 0$, soit $(x-4)^2 + (y-1)^2 = 9$. \checkmark.

\bigskip

\textbf{Exercice 4 — Position relative (4 pts)}

\textbf{1.} $d = \dfrac{|12 + 2 - 16|}{\sqrt{9+4}} = \dfrac{2}{\sqrt{13}} = \dfrac{2\sqrt{13}}{13} \approx 0{,}55$.

\textbf{2.} $d \approx 0{,}55 < 3 = r$ : la droite est sécante.

\textbf{3.} On pose $y = \dfrac{16 - 3x}{2}$ dans $(x-4)^2 + (y-1)^2 = 9$ :

$(x-4)^2 + \left(\dfrac{14-3x}{2}\right)^2 = 9$.

$4(x-4)^2 + (14-3x)^2 = 36$.

$4x^2 - 32x + 64 + 9x^2 - 84x + 196 = 36$, soit $13x^2 - 116x + 224 = 0$.

$\Delta = 13456 - 11648 = 1808$. $\sqrt{1808} = 4\sqrt{113}$.

$x = \dfrac{116 \pm 4\sqrt{113}}{26} = \dfrac{58 \pm 2\sqrt{113}}{13}$.

\bigskip

\textbf{Exercice 5 — Problème (4 pts)}

\textbf{1.} $\vec{AB}(4 ; -6)$, $\vec{A\Omega}(2 ; -4)$.

$\mathcal{A} = \dfrac{1}{2}|4 \times (-4) - (-6) \times 2| = \dfrac{1}{2}|-16 + 12| = \dfrac{4}{2} = 2$.

\textbf{2.} Perpendiculaire : $2x - 3y + c = 0$. En $\Omega(4;1)$ : $8 - 3 + c = 0$, $c = -5$.

$\begin{cases} 3x + 2y = 16 \\ 2x - 3y = 5 \end{cases}$. $9x + 6y = 48$ et $4x - 6y = 10$. $13x = 58$, $x = \dfrac{58}{13}$, $y = \dfrac{17}{13}$.

$H\!\left(\dfrac{58}{13} ; \dfrac{17}{13}\right)$.

\textbf{3.} $h = d(\Omega, (AB)) = \dfrac{2\sqrt{13}}{13}$.

Vérification : $AB = \sqrt{16+36} = 2\sqrt{13}$. $\mathcal{A} = \dfrac{1}{2} \times 2\sqrt{13} \times \dfrac{2\sqrt{13}}{13} = \dfrac{1}{2} \times \dfrac{4 \times 13}{13} = 2$ \checkmark.
