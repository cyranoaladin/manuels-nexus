\begin{exercice}{1SPE-PROBCOND-EX-049}{3}{20}
On effectue un double test de depistage. Le premier test a une sensibilite de $90\,\%$ et une specificite de $90\,\%$. Si le premier test est positif, on effectue un second test (sensibilite $95\,\%$, specificite $95\,\%$). La prevalence est de $2\,\%$.

\begin{enumerate}
  \item Calculer la probabilite d'un premier test positif.
  \item Calculer $P_T(M)$ pour le premier test seul.
  \item Calculer la probabilite que les deux tests soient positifs.
  \item Calculer la probabilite d'etre malade sachant que les deux tests sont positifs. Comparer avec la question 2.
\end{enumerate}
\end{exercice}
