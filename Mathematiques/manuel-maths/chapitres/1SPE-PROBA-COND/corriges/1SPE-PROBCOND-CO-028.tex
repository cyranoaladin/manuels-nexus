\textbf{1.} \textit{Demonstration.} $\{B, \overline{B}\}$ est une partition, donc $A = (A \cap B) \cup (A \cap \overline{B})$ avec evenements incompatibles. Par additivite : $P(A) = P(A \cap B) + P(A \cap \overline{B}) = P(B)\,P_B(A) + P(\overline{B})\,P_{\overline{B}}(A)$. $\square$

\textbf{2.} $P(A) = \frac{3}{8}\times\frac{2}{3} + \frac{5}{8}\times\frac{1}{4} = \frac{1}{4}+\frac{5}{32} = \frac{8}{32}+\frac{5}{32} = \frac{13}{32}$.
