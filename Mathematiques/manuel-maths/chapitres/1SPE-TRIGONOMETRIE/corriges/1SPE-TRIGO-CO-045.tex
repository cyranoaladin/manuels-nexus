\textbf{1.} $T(0) = 18 + 7 \times 1 = 25°$C. $T(6) = 18 + 7\cos\pi = 18 - 7 = 11°$C.

\textbf{2.} $\cos\!\left(\dfrac{\pi t}{6}\right)$ a pour periode $\dfrac{2\pi}{\pi/6} = 12$ heures.

\textbf{3.} Maximum : $18 + 7 = 25°$C. Minimum : $18 - 7 = 11°$C.

\textbf{4.} $T(t) = 18 \iff \cos\!\left(\dfrac{\pi t}{6}\right) = 0 \iff \dfrac{\pi t}{6} = \dfrac{\pi}{2} + k\pi$, soit $t = 3 + 6k$.

La temperature vaut $18°$C aux instants $t = 3$, $t = 9$, $t = 15$, etc. (toutes les $6$ heures a partir de $t = 3$).
