\begin{exercice}{1SPE-TRIGO-EX-016}{2}{12}
Pour tout reel $x$, simplifier les expressions suivantes.
\begin{enumerate}
  \item $\cos x + \cos(\pi - x)$
  \item $\sin x + \sin(\pi - x)$
  \item $\sin x \cdot \sin(\pi - x) + \cos x \cdot \cos(\pi - x)$
\end{enumerate}
\end{exercice}
