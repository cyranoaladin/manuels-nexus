\baremeIndicatif{Q1 : 4 pts — Q2 : 6 pts — Q3 : 5 pts — Q4 : 5 pts}

\medskip
\textbf{1. (C1) Forme canonique.}

\commentaireMarge{Compléter le carré : $x^2 - 4x = (x-2)^2 - 4$.}

$a = 1$, $b = -4$, $c = 1$.

\[\alpha = -\frac{b}{2a} = \frac{4}{2} = 2.\]

\[\beta = f(2) = 4 - 8 + 1 = -3.\]

\[\boxed{f(x) = (x - 2)^2 - 3.}\]

\textit{Vérification :} $(x-2)^2 - 3 = x^2 - 4x + 4 - 3 = x^2 - 4x + 1$. \hfill $\checkmark$

\medskip
\textbf{2. (C3) Résolution de $f(x) = 0$.}

\commentaireMarge{$\Delta = b^2 - 4ac$.}

$\Delta = (-4)^2 - 4 \times 1 \times 1 = 16 - 4 = 12 > 0$.

$\sqrt{\Delta} = \sqrt{12} = 2\sqrt{3}$.

\[x_1 = \frac{4 - 2\sqrt{3}}{2} = 2 - \sqrt{3}, \qquad x_2 = \frac{4 + 2\sqrt{3}}{2} = 2 + \sqrt{3}.\]

\[\boxed{\mathcal{S} = \left\{2 - \sqrt{3}\,;\,2 + \sqrt{3}\right\}.}\]

\medskip
\textbf{3. (C3) Résolution de $f(x) = -2$.}

\commentaireMarge{Ramener à $ax^2+bx+c=0$.}

\[f(x) = -2 \iff x^2 - 4x + 1 = -2 \iff x^2 - 4x + 3 = 0.\]

$a = 1$, $b = -4$, $c = 3$.

$\Delta = (-4)^2 - 4 \times 1 \times 3 = 16 - 12 = 4 > 0$, $\sqrt{\Delta} = 2$.

\[x_1 = \frac{4 - 2}{2} = 1, \qquad x_2 = \frac{4 + 2}{2} = 3.\]

\[\boxed{\mathcal{S} = \{1\,;\,3\}.}\]

\medskip
\textbf{4. (C1) La valeur $\beta = -3$ est-elle bien le minimum ?}

\commentaireMarge{$(x-2)^2 \geqslant 0$ pour tout $x \in \mathbb{R}$.}

Pour tout $x \in \mathbb{R}$, $(x-2)^2 \geqslant 0$, donc :
\[f(x) = (x-2)^2 - 3 \geqslant 0 - 3 = -3.\]

L'égalité est atteinte uniquement pour $x = 2$.

Ainsi, $f(x) \geqslant -3$ pour tout réel $x$, et $f(2) = -3$ : $\beta = -3$ est bien le \textbf{minimum} de $f$. \hfill $\square$
