\begin{corrige}{1SPE-SECDEG-EX-032}

\textbf{1. Système d'équations.}

\commentaireMarge{Substituer les coordonnées de chaque point dans $f(x) = ax^2 + bx + c$.}

\begin{itemize}
  \item $A(0\,;\,-3)$ : $f(0) = c = -3$.
  \item $B(1\,;\,0)$ : $f(1) = a + b + c = 0$.
  \item $C(-3\,;\,0)$ : $f(-3) = 9a - 3b + c = 0$.
\end{itemize}

Le système est :
\[\begin{cases} c = -3 \\ a + b + c = 0 \\ 9a - 3b + c = 0 \end{cases}\]

\medskip
\textbf{2. Résolution du système.}

\commentaireMarge{Substituer $c = -3$ dans les deux autres équations.}

De $c = -3$ et $a + b + c = 0$ : $a + b = 3$.

De $c = -3$ et $9a - 3b + c = 0$ : $9a - 3b = 3$, soit $3a - b = 1$.

En ajoutant $a + b = 3$ et $3a - b = 1$ :
\[4a = 4 \implies a = 1.\]
Puis $b = 3 - a = 2$.

Donc $f(x) = x^2 + 2x - 3$.

\medskip
\textbf{3. Forme canonique.}

\commentaireMarge{$\alpha = -\dfrac{b}{2a} = -\dfrac{2}{2} = -1$.}

\[\alpha = -\frac{2}{2} = -1, \qquad \beta = f(-1) = 1 - 2 - 3 = -4.\]

\[f(x) = (x + 1)^2 - 4.\]

\medskip
\textbf{4. Factorisation et vérification des racines.}

\commentaireMarge{Les racines sont les abscisses des points d'intersection avec l'axe des abscisses.}

Depuis la forme canonique : $\alpha = -1$, $\beta = -4 < 0$, racines :
\[x_{1,2} = -1 \pm \sqrt{4} = -1 \pm 2.\]
$x_1 = -3$ et $x_2 = 1$, donc $f(x) = (x + 3)(x - 1)$.

\textit{Vérification :} $B(1\,;\,0)$ car $f(1) = (1+3)(1-1) = 0$ ; $C(-3\,;\,0)$ car $f(-3) = 0$. \hfill $\square$
