\textbf{1. Étude de $f(x) = \left(x - \dfrac{3}{2}\right)^2 - \dfrac{25}{4}$.}

\textit{a) Développement.}

\commentaireMarge{Développer le carré, puis regrouper.}

\[f(x) = \left(x - \frac{3}{2}\right)^2 - \frac{25}{4} = x^2 - 3x + \frac{9}{4} - \frac{25}{4} = x^2 - 3x - \frac{16}{4} = x^2 - 3x - 4.\]

\textit{b) Factorisation.}

\commentaireMarge{Depuis la forme canonique : $x_{1,2} = \alpha \pm \sqrt{-\beta/a}$.}

La forme canonique est $f(x) = (x - \alpha)^2 + \beta$ avec $\alpha = \dfrac{3}{2}$ et $\beta = -\dfrac{25}{4}$.

Comme $a = 1$ et $\beta = -\dfrac{25}{4} < 0$, les racines sont :
\[x_{1,2} = \alpha \pm \sqrt{-\frac{\beta}{a}} = \frac{3}{2} \pm \sqrt{\frac{25}{4}} = \frac{3}{2} \pm \frac{5}{2}.\]

\[x_1 = \frac{3}{2} - \frac{5}{2} = -1 \qquad \text{et} \qquad x_2 = \frac{3}{2} + \frac{5}{2} = 4.\]

Donc $f(x) = (x + 1)(x - 4)$.

\textit{c) Tableau de signes.}

\commentaireMarge{$a = 1 > 0$ : positif à l'extérieur des racines.}

\hline
$f(x)$ & & $+$ & $0$ & $-$ & $0$ & $+$ & \\
\hline
\end{tabular}

\textbf{2. Étude de $g(x) = 4\!\left(x + \dfrac{1}{2}\right)^2 - 9$.}

\textit{a) Développement.}

\commentaireMarge{Développer le carré $\left(x + \frac{1}{2}\right)^2 = x^2 + x + \frac{1}{4}$.}

\[g(x) = 4\!\left(x^2 + x + \frac{1}{4}\right) - 9 = 4x^2 + 4x + 1 - 9 = 4x^2 + 4x - 8.\]

\textit{b) Factorisation et tableau de signes.}

\commentaireMarge{Mettre $4$ en facteur, puis trouver les racines.}

$g(x) = 4(x^2 + x - 2)$.

On cherche les racines de $x^2 + x - 2 = 0$ :
\[\Delta = 1 + 8 = 9, \qquad x_1 = \frac{-1 - 3}{2} = -2, \qquad x_2 = \frac{-1 + 3}{2} = 1.\]

Donc $g(x) = 4(x + 2)(x - 1)$.

\hline

\end{tabular}
\end{center}
