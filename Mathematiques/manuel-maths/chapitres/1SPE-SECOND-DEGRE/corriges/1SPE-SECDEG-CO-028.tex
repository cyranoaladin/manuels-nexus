\textbf{1. Coordonnées du sommet.}

\commentaireMarge{$\alpha = -\dfrac{b}{2a}$ avec $a=-2$, $b=80$.}

$B(x) = -2x^2 + 80x - 600$ : on a $a = -2$, $b = 80$, $c = -600$.

\[\alpha = -\frac{80}{2 \times (-2)} = \frac{80}{4} = 20.\]

\[\beta = B(20) = -2(400) + 80(20) - 600 = -800 + 1600 - 600 = 200.\]

Le sommet est $S(20\,;\,200)$.

\medskip

\commentaireMarge{$a = -2 < 0$ : parabole vers le bas, maximum en $\alpha = 20$.}

\textbf{3. Maximisation du bénéfice.}

\commentaireMarge{Le maximum est atteint au sommet de la parabole.}

Le bénéfice maximal est $B(20) = \mathbf{200}$~\euro.

\textbf{4. Intervalle de rentabilité.}

\commentaireMarge{Résoudre $B(x) = 0$, puis lire le signe.}

$B(x) = 0 \iff -2x^2 + 80x - 600 = 0 \iff x^2 - 40x + 300 = 0$.

\[\Delta = 1600 - 1200 = 400.\]

\[x_1 = \frac{40 - 20}{2} = 10 \qquad \text{et} \qquad x_2 = \frac{40 + 20}{2} = 30.\]

Comme $a = -2 < 0$, $B(x) > 0$ entre les racines : l'entreprise est rentable pour $x \in \left]10\,;\,30\right[$.
