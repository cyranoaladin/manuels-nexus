\begin{corrige}{1SPE-SECDEG-EX-022}

\textbf{1. Forme canonique de $p(x)$.}

\commentaireMarge{Mettre en facteur $a$, puis compléter le carré.}

On a $p(x) = 2x^2 - 4x - 6$, avec $a = 2$, $b = -4$, $c = -6$.

L'abscisse du sommet est :
\[\alpha = -\frac{b}{2a} = -\frac{-4}{2 \times 2} = 1.\]

L'ordonnée du sommet est :
\[\beta = p(1) = 2(1)^2 - 4(1) - 6 = 2 - 4 - 6 = -8.\]

Donc :
\[p(x) = 2(x - 1)^2 - 8.\]

\medskip
\textbf{1. Forme factorisée de $p(x)$.}

\commentaireMarge{$\Delta = b^2 - 4ac$ ; racines $x_{1,2} = \dfrac{-b \pm \sqrt{\Delta}}{2a}$.}

On calcule le discriminant :
\[\Delta = (-4)^2 - 4 \times 2 \times (-6) = 16 + 48 = 64 > 0.\]

Les deux racines sont :
\[x_1 = \frac{4 - \sqrt{64}}{4} = \frac{4 - 8}{4} = -1 \qquad \text{et} \qquad x_2 = \frac{4 + 8}{4} = 3.\]

D'où :
\[p(x) = 2(x + 1)(x - 3).\]

\medskip
\textbf{2. Résolution de $p(x) = 10$.}

\commentaireMarge{Ramener à $p(x) - 10 = 0$.}

$p(x) = 10 \iff 2x^2 - 4x - 6 = 10 \iff 2x^2 - 4x - 16 = 0 \iff x^2 - 2x - 8 = 0$.

Discriminant : $\Delta' = (-2)^2 - 4 \times 1 \times (-8) = 4 + 32 = 36$.

\[x_1 = \frac{2 - 6}{2} = -2 \qquad \text{et} \qquad x_2 = \frac{2 + 6}{2} = 4.\]

L'ensemble des solutions est $\mathcal{S} = \{-2\,;\,4\}$.

\medskip
\textbf{3. Résolution de $p(x) = -8$.}

\commentaireMarge{Utiliser la forme canonique : le minimum de $p$ vaut $-8$.}

D'après la forme canonique : $p(x) = 2(x-1)^2 - 8$.

$p(x) = -8 \iff 2(x-1)^2 - 8 = -8 \iff 2(x-1)^2 = 0 \iff (x-1)^2 = 0 \iff x = 1$.

L'ensemble des solutions est $\mathcal{S} = \{1\}$.

\medskip
\textbf{4. Vérification pour $x = 2$.}

\commentaireMarge{Les trois formes doivent donner la même valeur.}

\begin{itemize}
  \item Forme développée : $p(2) = 2(4) - 4(2) - 6 = 8 - 8 - 6 = -6$.
  \item Forme canonique : $p(2) = 2(2-1)^2 - 8 = 2 \times 1 - 8 = -6$.
  \item Forme factorisée : $p(2) = 2(2+1)(2-3) = 2 \times 3 \times (-1) = -6$.
\end{itemize}

Les trois formes donnent bien $p(2) = -6$. \hfill $\square$
