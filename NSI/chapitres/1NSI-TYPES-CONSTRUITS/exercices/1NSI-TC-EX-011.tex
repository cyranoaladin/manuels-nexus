\begin{exercice}{1NSI-TC-EX-011}{2}{12}
On modélise les résultats d'un tournoi par une liste de tuples
\lstinline{(equipe, score_pour, score_contre)}.

Écrire une fonction \lstinline{classer_matchs(matchs)} qui renvoie un tuple de trois
listes : les équipes victorieuses, les équipes perdantes et les matchs nuls.

\textbf{Exemple :}
\begin{console}
>>> classer_matchs([("A", 3, 1), ("B", 0, 2), ("C", 1, 1)])
(['A'], ['B'], ['C'])
\end{console}
\end{exercice}
