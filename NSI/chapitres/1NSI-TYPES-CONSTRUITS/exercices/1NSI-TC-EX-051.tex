\begin{exercice}{1NSI-TC-EX-051}{2}{10}
Un élève écrit le code suivant pour supprimer les valeurs négatives d'un tableau :
\begin{python}
def supprimer_negatifs_v1(tab):
    for val in tab:
        if val < 0:
            tab.remove(val)
    return tab
\end{python}
\begin{enumerate}
  \item Tester mentalement \lstinline{supprimer_negatifs_v1([5, -2, -3, 8])}.
        Le résultat est-il correct ? Expliquer le problème.
  \item Voici une version corrigée :
\begin{python}
def supprimer_negatifs(tab):
    resultat = []
    for val in tab:
        if val >= 0:
            resultat.append(val)
    return resultat
\end{python}
        Pourquoi cette version est-elle plus fiable ? Le tableau d'origine
        est-il modifié ?
\end{enumerate}
\end{exercice}
