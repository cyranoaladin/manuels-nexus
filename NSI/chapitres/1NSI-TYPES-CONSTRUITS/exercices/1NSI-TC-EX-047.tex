\begin{exercice}{1NSI-TC-EX-047}{1}{5}
On considère le programme suivant :
\begin{python}
a = [1, 2, 3]
b = list(a)
b.append(4)
print(a)
print(b)
print(a is b)
\end{python}
\begin{enumerate}
  \item Donner, sans exécuter, les trois affichages.
  \item Quelle est la différence avec \lstinline{b = a} ?
  \item Citer deux autres façons de créer une copie superficielle d'une liste.
\end{enumerate}
\end{exercice}
