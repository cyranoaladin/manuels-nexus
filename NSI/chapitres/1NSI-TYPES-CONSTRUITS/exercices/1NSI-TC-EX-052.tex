\begin{exercice}{1NSI-TC-EX-052}{2}{10}
On considère le programme suivant :
\begin{python}
equipe = {"nom": "Phoenix", "score": 100}
copie = equipe
copie["score"] = 200
print(equipe["score"])
print(equipe is copie)
\end{python}
\begin{enumerate}
  \item Donner, sans exécuter, les deux affichages.
  \item Les dictionnaires sont-ils mutables ? Justifier.
  \item Réécrire le code pour que la modification de \lstinline{copie} ne modifie
        pas \lstinline{equipe}. On pourra utiliser \lstinline{dict(equipe)}.
\end{enumerate}
\end{exercice}
