\begin{exercice}{1NSI-TC-EX-001}{1}{8}
On considère le programme suivant :
\begin{python}
notes = [12, 15, 9]
copie = notes
copie.append(18)
print(notes)
print(len(copie))
\end{python}
\begin{enumerate}
  \item Donner, sans exécuter le programme, ce qu'affichent les deux instructions \texttt{print}.
  \item Expliquer pourquoi la liste \texttt{notes} est modifiée alors que l'on n'a
        appelé \texttt{append} que sur \texttt{copie}.
  \item Modifier la ligne 2 pour que \texttt{notes} ne soit pas modifiée.
\end{enumerate}
\end{exercice}
