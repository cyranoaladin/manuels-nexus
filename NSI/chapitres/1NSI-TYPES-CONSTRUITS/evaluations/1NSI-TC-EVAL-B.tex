\textbf{Exercice 1} --- \textit{Analyse de programme} \hfill (5 points)

On donne le programme suivant :
\begin{python}

panier = []
for fruit, quantite in inventaire.items():
    if quantite > 6:
        panier.append((fruit, quantite))
inventaire["pommes"] = 0
print(panier)
print(inventaire["pommes"])
\end{python}
