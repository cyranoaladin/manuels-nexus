\subsection{Exercice 1 - Fonction renvoyant un p-uplet (P-DATA-CONSTR-01)}

On considère la fonction suivante :

\begin{python}
def analyser_notes(notes):
    mini = notes[0]
    maxi = notes[0]
    somme = 0
    for n in notes:
        if n < mini:
            mini = n
        if n > maxi:
            maxi = n
        somme = somme + n
    moyenne = somme / len(notes)
    return mini, maxi, moyenne
\end{python}

\textbf{Question 1a.} Donner le type de la valeur renvoyée par
\texttt{analyser\_notes({[}12,\ 8,\ 15,\ 10{]})}.

\textbf{Question 1b.} Déterminer les valeurs de \texttt{a}, \texttt{b}
et \texttt{c} après l'exécution de :

\begin{python}
a, b, c = analyser_notes([12, 8, 15, 10])
\end{python}

\textbf{Question 1c.} Écrire sur papier une fonction
\texttt{extremes(a,\ b)} qui renvoie le tuple
\texttt{(plus\_petit,\ plus\_grand)} de deux nombres \texttt{a} et
\texttt{b}.

\textbf{Question 1d.} Que se passe-t-il si on appelle
\texttt{analyser\_notes({[}{]})} ? Quelle précondition faudrait-il
vérifier ?
