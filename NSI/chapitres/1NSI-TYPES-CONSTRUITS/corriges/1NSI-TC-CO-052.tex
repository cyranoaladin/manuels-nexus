\begin{corrige}{1NSI-TC-EX-052}
\begin{enumerate}
  \item Le programme affiche :
\begin{console}
200
True
\end{console}

  \item Oui, les dictionnaires sont \textbf{mutables}. On peut ajouter, modifier ou supprimer des paires clé-valeur après la création. Ici, \lstinline{copie["score"] = 200} modifie le dictionnaire en place. Comme \lstinline{copie = equipe} crée un alias (même objet), la modification est visible via \lstinline{equipe} aussi.

  \item Pour obtenir une copie indépendante, on utilise \lstinline{dict()} :
\begin{python}
equipe = {"nom": "Phoenix", "score": 100}
copie = dict(equipe)
copie["score"] = 200
print(equipe["score"])  # 100
print(copie["score"])   # 200
\end{python}
Après cette modification, \lstinline{equipe["score"]} vaut toujours \lstinline{100} car \lstinline{copie} est un objet distinct.
\end{enumerate}
\end{corrige}
