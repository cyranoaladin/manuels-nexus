\begin{corrige}{1NSI-TC-EX-039}
On parcourt le tableau et, pour chaque élément, on teste s'il est déjà dans le dictionnaire. Si oui, on incrémente son compteur ; sinon, on l'initialise à 1.

\begin{python}
def compter_occurrences(tab):
    compteur = {}
    for element in tab:
        if element in compteur:
            compteur[element] = compteur[element] + 1
        else:
            compteur[element] = 1
    return compteur
\end{python}

\textbf{Déroulement sur l'exemple} \lstinline{["a", "b", "a", "c", "b", "a"]} :
\begin{itemize}
  \item \lstinline{"a"} absent $\to$ \lstinline{compteur = {"a": 1}}
  \item \lstinline{"b"} absent $\to$ \lstinline{compteur = {"a": 1, "b": 1}}
  \item \lstinline{"a"} présent $\to$ \lstinline{compteur = {"a": 2, "b": 1}}
  \item \lstinline{"c"} absent $\to$ \lstinline{compteur = {"a": 2, "b": 1, "c": 1}}
  \item \lstinline{"b"} présent $\to$ \lstinline{compteur = {"a": 2, "b": 2, "c": 1}}
  \item \lstinline{"a"} présent $\to$ \lstinline{compteur = {"a": 3, "b": 2, "c": 1}}
\end{itemize}
\end{corrige}
