\begin{corrige}{1NSI-TC-EX-009}
\textbf{1.} On parcourt la liste en déballant chaque tuple et on conserve ceux dont le score Elo atteint le seuil :

\begin{python}
def filtrer_par_rang(joueurs, rang_min):
    resultat = []
    for nom, elo, rang in joueurs:
        # On compare le score Elo au seuil
        if elo >= rang_min:
            resultat.append((nom, elo, rang))
    return resultat
\end{python}

Le déballage \lstinline{for nom, elo, rang in joueurs} affecte directement les trois composantes de chaque tuple aux variables \lstinline{nom}, \lstinline{elo} et \lstinline{rang}, ce qui rend le code lisible.

\textbf{2.} Test avec la liste donnée :
\begin{console}
>>> filtrer_par_rang(joueurs, 2000)
[('Alice', 2400, 'diamant'), ('Clara', 2100, 'platine')]
\end{console}

Bob (Elo 1800) est exclu car $1800 < 2000$.

Vérification : les résultats ont été confirmés par exécution.
\end{corrige}
