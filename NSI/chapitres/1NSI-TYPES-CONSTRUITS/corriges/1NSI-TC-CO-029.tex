\begin{corrige}{1NSI-TC-EX-029}
On initialise le maximum avec la première case, puis on parcourt toute la grille en mettant à jour la position et la valeur du maximum :

\begin{python}
def maximum_grille(grille):
    maxi = grille[0][0]
    ligne_max = 0
    col_max = 0
    for i in range(len(grille)):
        for j in range(len(grille[i])):
            if grille[i][j] > maxi:
                maxi = grille[i][j]
                ligne_max = i
                col_max = j
    return (ligne_max, col_max, maxi)
\end{python}

\begin{console}
>>> maximum_grille([[1, 5], [9, 3]])
(1, 0, 9)
\end{console}

On utilise \lstinline{range(len(...))} pour disposer des indices \lstinline{i} et \lstinline{j}, nécessaires au renvoi de la position.
\end{corrige}
