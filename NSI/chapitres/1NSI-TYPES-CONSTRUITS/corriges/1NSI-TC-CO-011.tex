\begin{corrige}{1NSI-TC-EX-011}
On crée trois listes accumulatrices et on classe chaque équipe selon la comparaison des scores :

\begin{python}
def classer_matchs(matchs):
    victoires = []
    defaites = []
    nuls = []
    for equipe, score_pour, score_contre in matchs:
        if score_pour > score_contre:
            victoires.append(equipe)
        elif score_pour < score_contre:
            defaites.append(equipe)
        else:
            nuls.append(equipe)
    return (victoires, defaites, nuls)
\end{python}

Le déballage \lstinline{for equipe, score_pour, score_contre in matchs} extrait directement les trois composantes de chaque tuple. La structure conditionnelle \lstinline{if / elif / else} couvre les trois cas possibles (victoire, défaite, match nul). La fonction renvoie un tuple de trois listes.

Vérification : \lstinline{classer_matchs([("A", 3, 1), ("B", 0, 2), ("C", 1, 1)])} renvoie bien \lstinline{(["A"], ["B"], ["C"])}.
\end{corrige}
