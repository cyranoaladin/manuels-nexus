\begin{corrige}{1SPE-SECDEG-EX-038}

\textbf{1. Discriminant en fonction de $m$.}

\textit{a) Calcul de $\Delta(m)$.}

\commentaireMarge{$\Delta = b^2 - 4ac$ avec $b = -2m$, $c = m^2 - m - 2$.}

\[\Delta(m) = (-2m)^2 - 4 \times 1 \times (m^2 - m - 2) = 4m^2 - 4m^2 + 4m + 8 = 4m + 8.\]

\textit{b) Signe de $\Delta(m)$ et nombre de racines.}

\commentaireMarge{Étudier le signe de $4m + 8 = 4(m+2)$.}

$\Delta(m) = 4(m + 2)$.
\begin{itemize}
  \item Si $m < -2$ : $\Delta < 0$, pas de racine réelle.
  \item Si $m = -2$ : $\Delta = 0$, une racine double.
  \item Si $m > -2$ : $\Delta > 0$, deux racines réelles distinctes.
\end{itemize}

\medskip
\textbf{2. Cas particuliers.}

\textit{a) Pour $m = 0$.}

\commentaireMarge{$f_0(x) = x^2 - 2$.}

$f_0(x) = x^2 - 2$ : $\Delta = 8$, donc $x_{1,2} = \pm\sqrt{2}$.

\textit{b) Pour $m = -2$.}

\commentaireMarge{$\Delta(-2) = 0$ : racine double.}

$f_{-2}(x) = x^2 + 4x + (4 + 2 - 2) = x^2 + 4x + 4 = (x+2)^2$.

Racine double : $x_0 = -2$. On vérifie : $f_{-2}(-2) = 0$. \hfill $\square$

\textit{c) Pour $m = 2$.}

$f_2(x) = x^2 - 4x + (4 - 2 - 2) = x^2 - 4x$. $\Delta = 16$, racines :

\[x_1 = \frac{4 - 4}{2} = 0 \qquad \text{et} \qquad x_2 = \frac{4 + 4}{2} = 4.\]

La racine $x_1 = 0$ est remarquable.

\medskip
\textbf{3. Forme canonique.}

\textit{a) Forme canonique de $f_m$.}

\commentaireMarge{$\alpha = \dfrac{2m}{2} = m$.}

\[\alpha = m, \qquad \beta = f_m(m) = m^2 - 2m^2 + m^2 - m - 2 = -m - 2.\]

\[f_m(x) = (x - m)^2 - (m + 2).\]

\textit{b) Sommet en fonction de $m$.}

Le sommet est $S(m\,;\,-m-2)$ : ses coordonnées dépendent toutes deux de $m$. Lorsque $m$ varie, le sommet se déplace le long de la droite d'équation $\beta = -\alpha - 2$.

\medskip
\textbf{4. Valeurs de $m$ pour lesquelles $0$ est racine.}

\textit{a) Condition.}

\commentaireMarge{$f_m(0) = 0 \iff m^2 - m - 2 = 0$.}

$f_m(0) = m^2 - m - 2 = 0$.

$\Delta_m = 1 + 8 = 9$, donc $m = \dfrac{1 \pm 3}{2}$, c'est-à-dire $m = -1$ ou $m = 2$.

\textit{b) Factorisation pour ces valeurs.}

Pour $m = -1$ : $f_{-1}(x) = x^2 + 2x + 0 = x(x + 2)$.

Pour $m = 2$ : $f_2(x) = x^2 - 4x + 0 = x(x - 4)$.

\medskip
\textbf{5. Question ouverte : deux racines strictement positives.}

\commentaireMarge{Conditions : $\Delta > 0$, somme des racines $> 0$ et produit des racines $> 0$.}

Pour $f_m(x) = x^2 - 2mx + (m^2 - m - 2)$, les relations coefficients-racines donnent :
\[x_1 + x_2 = 2m \quad \text{et} \quad x_1 \cdot x_2 = m^2 - m - 2.\]

Conditions pour deux racines strictement positives :
\begin{enumerate}
  \item $\Delta(m) > 0 \iff m > -2$.
  \item $x_1 + x_2 > 0 \iff 2m > 0 \iff m > 0$.
  \item $x_1 \cdot x_2 > 0 \iff m^2 - m - 2 > 0 \iff (m-2)(m+1) > 0 \iff m < -1$ ou $m > 2$.
\end{enumerate}

L'intersection des trois conditions : $m > -2$ et $m > 0$ et ($m < -1$ ou $m > 2$).

La condition $m > 0$ exclut $m < -1$, donc : $m > 0$ et $m > 2$, soit $m > 2$.

$f_m$ admet deux racines strictement positives si et seulement si $m > 2$.
