\begin{corrige}{1SPE-SECDEG-EX-033}

\textbf{1. Quantité optimale et profit maximal.}

\commentaireMarge{Sommet de la parabole : $\alpha = -\dfrac{b}{2a}$.}

$P(x) = -x^2 + 14x - 40$ : on a $a = -1$, $b = 14$, $c = -40$.

\[\alpha = -\frac{14}{2 \times (-1)} = 7.\]

\[\beta = P(7) = -49 + 98 - 40 = 9.\]

La quantité optimale est $x = 7$ (centaines d'unités), et le profit maximal est $\mathbf{9\,000}$~\euro.

\medskip
\textbf{2. Intervalle de rentabilité ($P(x) > 0$).}

\commentaireMarge{Résoudre $P(x) = 0$, puis lire le signe avec $a < 0$.}

$P(x) = 0 \iff x^2 - 14x + 40 = 0$.

\[\Delta = 196 - 160 = 36.\]

\[x_1 = \frac{14 - 6}{2} = 4 \qquad \text{et} \qquad x_2 = \frac{14 + 6}{2} = 10.\]

Comme $a = -1 < 0$, $P(x) > 0$ entre les racines : l'entreprise est rentable pour $x \in \left]4\,;\,10\right[$.

\medskip
\textbf{3. Condition $P(x) \geqslant 5$.}

\textit{a) Reformulation.}

\commentaireMarge{Ramener à une inéquation du second degré.}

$P(x) \geqslant 5 \iff -x^2 + 14x - 40 \geqslant 5 \iff -x^2 + 14x - 45 \geqslant 0$. \hfill $\square$

\textit{b) Résolution de $-x^2 + 14x - 45 \geqslant 0$.}

\commentaireMarge{$a = -1 < 0$ : positif entre les racines.}

On pose $Q(x) = -x^2 + 14x - 45$, avec $a = -1$, $b = 14$, $c = -45$.

\[\Delta = 196 - 4 \times (-1) \times (-45) = 196 - 180 = 16.\]

\[x_1 = \frac{-14 + 4}{-2} = 5 \qquad \text{et} \qquad x_2 = \frac{-14 - 4}{-2} = 9.\]

Comme $a = -1 < 0$, $Q(x) \geqslant 0$ pour $x \in [5\,;\,9]$.

\textit{Interprétation :} pour obtenir un profit d'au moins $5\,000$~\euro, l'entreprise doit produire entre $500$ et $900$ unités par mois.
