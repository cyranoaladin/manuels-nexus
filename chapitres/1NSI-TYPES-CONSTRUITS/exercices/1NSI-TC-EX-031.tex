\begin{exercice}{1NSI-TC-EX-031}{2}{12}
Dans une grille binaire (contenant des 0 et des 1), on veut compter les voisins
d'une case.

Écrire une fonction \lstinline{compter_voisins(grille, lig, col)} qui renvoie le
nombre de cases voisines (horizontales, verticales et diagonales) contenant la
valeur 1, sans compter la case elle-même.

\textbf{Exemple :}
\begin{console}
>>> g = [[0, 1, 0], [1, 1, 0], [0, 0, 1]]
>>> compter_voisins(g, 1, 1)
2
>>> compter_voisins(g, 0, 0)
2
\end{console}

\textit{Attention :} gérer les bords de la grille.
\end{exercice}
