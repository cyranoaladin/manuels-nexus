\begin{exercice}{1NSI-TC-EX-017}{1}{8}
On considère le programme suivant :
\begin{python}
notes = [12, 8, 15, 6, 14, 9]
tab = [n for n in notes if n >= 10]
print(tab)
doubles = [n * 2 for n in [1, 2, 3]]
print(doubles)
\end{python}
\begin{enumerate}
  \item Donner, sans exécuter, les deux affichages.
  \item Réécrire la construction de \lstinline{tab} en utilisant une boucle
        \lstinline{for} et \lstinline{append} (sans compréhension de liste).
\end{enumerate}
\end{exercice}
