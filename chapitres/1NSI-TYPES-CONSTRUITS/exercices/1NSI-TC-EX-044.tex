\begin{exercice}{1NSI-TC-EX-044}{3}{15}
On modélise un tournoi d'e-sport par une liste de tuples
\lstinline{(equipe1, score1, equipe2, score2)}.

Écrire une fonction \lstinline{classement_tournoi(matchs)} qui renvoie un
dictionnaire associant à chaque équipe son total de points : 3 points pour une
victoire, 1 point pour un match nul, 0 pour une défaite.

\textbf{Exemple :}
\begin{console}
>>> matchs = [("A", 2, "B", 1), ("A", 0, "C", 0), ("B", 3, "C", 1)]
>>> classement_tournoi(matchs)
{'A': 4, 'B': 3, 'C': 1}
\end{console}
\end{exercice}
