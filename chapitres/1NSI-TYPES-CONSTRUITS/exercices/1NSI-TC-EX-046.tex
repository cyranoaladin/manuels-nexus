\begin{exercice}{1NSI-TC-EX-046}{1}{5}
On considère le programme suivant :
\begin{python}
a = [1, 2, 3]
b = a
b.append(4)
print(a)
print(a is b)
\end{python}
\begin{enumerate}
  \item Donner, sans exécuter, les deux affichages.
  \item Pourquoi la liste \lstinline{a} est-elle modifiée alors qu'on n'a appelé
        \lstinline{append} que sur \lstinline{b} ?
  \item Que signifie \lstinline{a is b} ?
\end{enumerate}
\end{exercice}
