\subsection{Erreurs fréquentes}

\begin{itemize}
\tightlist
\item
  Confondre le vocabulaire de listes avec une simple récitation :
  corriger par un exemple calculé ou exécuté.
\item
  Oublier une hypothèse de tuples, listes et dictionnaires : corriger en
  l'écrivant avant la méthode.
\item
  Conclure sans contrôle sur types construits : corriger par un cas
  limite ou une vérification inverse.
\end{itemize}

\subsection{Cas limites}

\begin{itemize}
\tightlist
\item
  Cas de départ vide ou nul pour listes, à traiter selon la convention
  du chapitre P04.
\item
  Donnée invalide dans tuples, listes et dictionnaires, par exemple
  symbole interdit, clé absente ou requête trop large selon la fiche.
\item
  Cas frontière de types construits où une seule valeur change la
  méthode ou le résultat attendu.
\end{itemize}

\def\labelenumi{\arabic{enumi}.}

\subsection{Complément types construits P04}

\begin{itemize}
\tightlist
\item
  Tuple immuable : \texttt{point\ =\ (3.0,\ 5.0)} représente une
  coordonnée fixe ; \texttt{point{[}0{]}\ =\ 4.0} doit être refusé car
  un tuple ne se modifie pas.
\item
  Liste mutable : \texttt{notes\ =\ {[}8,\ 12,\ 10{]}}, puis
  \texttt{notes{[}1{]}\ =\ 14} donne \texttt{{[}8,\ 14,\ 10{]}}.
\item
  Dictionnaire par clé :
  \texttt{station\ =\ \{"nom":\ "Carthage",\ "temperature":\ 21\}} ;
  \texttt{station{[}"temperature"{]}} vaut \texttt{21}.
\item
  Liste de dictionnaires :
  \texttt{stations\ =\ {[}\{"nom":\ "A",\ "temperature":\ 21\},\ \{"nom":\ "B",\ "temperature":\ 18\}{]}}
  permet de filtrer les stations dont
  \texttt{temperature\ \textgreater{}=\ 20}.
\item
  Tuple de coordonnées : pour \texttt{A\ =\ (2,\ 3)} et
  \texttt{B\ =\ (8,\ 7)}, le milieu vaut
  \texttt{((2+8)/2,\ (3+7)/2)\ =\ (5.0,\ 5.0)}.
\item
  Distance : avec les mêmes points, \texttt{dx\ =\ 6},
  \texttt{dy\ =\ 4}, donc
  \texttt{distance\ =\ sqrt(6**2\ +\ 4**2)\ =\ sqrt(52)}.
\item
  Cas limite tuple : \texttt{(2,)} n'a pas deux coordonnées ; une
  fonction \texttt{milieu(A,\ B)} doit lever \texttt{ValueError} si
  \texttt{len(A)\ !=\ 2}.
\item
  Cas limite dictionnaire : \texttt{station{[}"pression"{]}} lève
  \texttt{KeyError}; on utilise \texttt{"pression"\ in\ station} ou
  \texttt{station.get("pression")}.
\item
  Cas limite liste : la moyenne d'une liste vide n'est pas définie ; le
  code doit tester \texttt{if\ not\ notes}.
\end{itemize}

\subsection{Mini-exercices corrigés complémentaires}

\subsubsection{Mini-exercice 5}

Calculer le milieu de \texttt{A=(0,\ 4)} et \texttt{B=(6,\ 10)}.

Réponse : \texttt{((0+6)/2,\ (4+10)/2)\ =\ (3.0,\ 7.0)}.

\subsubsection{Mini-exercice 6}

Après
\texttt{mesures\ =\ {[}\{"jour":\ "lundi",\ "temperature":\ 20\},\ \{"jour":\ "mardi",\ "temperature":\ 17\}{]}},
donner les jours de température au moins 18.

Réponse : seul \texttt{"lundi"} est gardé, car
\texttt{20\ \textgreater{}=\ 18} et \texttt{17\ \textless{}\ 18}.

\subsubsection{Mini-exercice 7}

Dire pourquoi \texttt{point{[}1{]}\ =\ 9} est incorrect si
\texttt{point\ =\ (2,\ 3)}.

Réponse : \texttt{point} est un tuple immuable ; il faut créer un
nouveau tuple, par exemple \texttt{(2,\ 9)}.
