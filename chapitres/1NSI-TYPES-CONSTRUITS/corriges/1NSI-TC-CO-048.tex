\begin{corrige}{1NSI-TC-EX-048}
\begin{enumerate}
  \item Le programme affiche :
\begin{console}
[10, 20, 30]
\end{console}

  \item Lorsqu'on passe une liste à une fonction, le paramètre \lstinline{tab} reçoit une \textbf{référence} vers le même objet que \lstinline{scores}. L'appel \lstinline{tab.append(val)} modifie donc directement la liste d'origine, sans qu'un \lstinline{return} soit nécessaire.

  \item Un \textbf{effet de bord} est une modification d'un état extérieur à la fonction (ici la liste \lstinline{scores}) produite par l'exécution de la fonction. La fonction ne se contente pas de calculer une valeur : elle modifie un objet mutable partagé. Les effets de bord rendent le code plus difficile à comprendre car le résultat ne dépend pas uniquement de la valeur de retour.
\end{enumerate}
\end{corrige}
