\begin{corrige}{1NSI-TC-EX-041}
On utilise deux dictionnaires auxiliaires : \lstinline{totaux} pour la somme des notes et \lstinline{effectifs} pour le nombre d'élèves par classe. On calcule ensuite chaque moyenne.

\begin{python}
def moyenne_par_classe(eleves):
    totaux = {}
    effectifs = {}
    for e in eleves:
        c = e["classe"]
        if c not in totaux:
            totaux[c] = 0
            effectifs[c] = 0
        totaux[c] = totaux[c] + e["note"]
        effectifs[c] = effectifs[c] + 1
    moyennes = {}
    for c in totaux:
        moyennes[c] = totaux[c] / effectifs[c]
    return moyennes
\end{python}

\textbf{Vérification :}
\begin{console}
>>> moyenne_par_classe(eleves)
{'1A': 13.0, '1B': 15.0}
\end{console}

Pour la classe \lstinline{"1A"}, la somme vaut $12 + 14 = 26$ et l'effectif vaut 2, d'où une moyenne de $26 / 2 = 13{,}0$.
\end{corrige}
