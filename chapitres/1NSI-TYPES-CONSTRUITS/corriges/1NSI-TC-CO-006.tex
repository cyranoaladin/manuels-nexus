\begin{corrige}{1NSI-TC-EX-006}
\begin{python}
def coordonnees_milieu(a, b):
    # a et b sont des tuples (x, y)
    return ((a[0] + b[0]) / 2, (a[1] + b[1]) / 2)
\end{python}

La fonction accède aux coordonnées par indexation (\lstinline{a[0]} pour $x_A$, \lstinline{a[1]} pour $y_A$) et renvoie un nouveau tuple contenant les moyennes.

Vérification : \lstinline{coordonnees_milieu((0, 0), (4, 6))} renvoie bien \lstinline{(2.0, 3.0)}.
\end{corrige}
