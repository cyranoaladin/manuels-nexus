\begin{python}
def moyenne(tab):
    somme = 0
    for val in tab:
        somme = somme + val
    return somme / len(tab)
\end{python}

On parcourt le tableau pour accumuler la somme des éléments, puis on divise par le nombre d'éléments (\lstinline{len(tab)}). La division \lstinline{/} renvoie un flottant.
