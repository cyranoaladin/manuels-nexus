def fusionner(tab1, tab2):
    resultat = []
    i = 0  # indice de parcours de tab1
    j = 0  # indice de parcours de tab2
    # Tant que les deux tableaux ont des elements a traiter
    while i < len(tab1) and j < len(tab2):
        if tab1[i] <= tab2[j]:
            resultat.append(tab1[i])
            i = i + 1
        else:
            resultat.append(tab2[j])
            j = j + 1
    # On ajoute les elements restants de tab1
    while i < len(tab1):
        resultat.append(tab1[i])
        i = i + 1
    # On ajoute les elements restants de tab2
    while j < len(tab2):
        resultat.append(tab2[j])
        j = j + 1
    return resultat
\end{python}

\textbf{Principe :} on utilise deux indices \lstinline{i} et \lstinline{j} parcourant respectivement \lstinline{tab1} et \lstinline{tab2}. À chaque étape, on compare les éléments courants et on ajoute le plus petit au résultat. Quand l'un des tableaux est entièrement parcouru, on ajoute les éléments restants de l'autre.

Le test \lstinline{<=} (plutôt que \lstinline{<}) assure la \textbf{stabilité} : en cas d'égalité, l'élément de \lstinline{tab1} est ajouté en premier.

Vérification : \lstinline{fusionner([1, 3, 5], [2, 4, 6])} renvoie \lstinline{[1, 2, 3, 4, 5, 6]}. Résultat confirmé par exécution.
