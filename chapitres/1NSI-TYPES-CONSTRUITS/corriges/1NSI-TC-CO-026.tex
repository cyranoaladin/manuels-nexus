  En effet, la double boucle parcourt toutes les valeurs de la grille et les accumule dans \lstinline{total} : $20+22+19+25+28+23+18+17+21 = 193$.

  \item Pour afficher la moyenne, on divise le total par le nombre de cases. La grille contient $3 \times 3 = 9$ valeurs :
\begin{python}
carte = [[20, 22, 19],
         [25, 28, 23],
         [18, 17, 21]]
total = 0
nb = 0
for ligne in carte:
    for val in ligne:
        total = total + val
        nb = nb + 1
print(total / nb)
\end{python}
  L'affichage est \lstinline{21.444444444444443}.
