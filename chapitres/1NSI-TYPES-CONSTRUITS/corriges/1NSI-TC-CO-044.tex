\begin{corrige}{1NSI-TC-EX-044}
On parcourt chaque match en déballant le tuple. Pour chaque équipe rencontrée pour la première fois, on initialise son compteur à 0. Puis on attribue les points selon le résultat.

\begin{python}
def classement_tournoi(matchs):
    points = {}
    for equipe1, score1, equipe2, score2 in matchs:
        if equipe1 not in points:
            points[equipe1] = 0
        if equipe2 not in points:
            points[equipe2] = 0
        if score1 > score2:
            points[equipe1] = points[equipe1] + 3
        elif score1 < score2:
            points[equipe2] = points[equipe2] + 3
        else:
            points[equipe1] = points[equipe1] + 1
            points[equipe2] = points[equipe2] + 1
    return points
\end{python}

\textbf{Déroulement sur l'exemple :}
\begin{itemize}
  \item \lstinline{("A", 2, "B", 1)} : A gagne ($2 > 1$) $\to$ A reçoit 3 points.
  \item \lstinline{("A", 0, "C", 0)} : match nul $\to$ A et C reçoivent chacun 1 point.
  \item \lstinline{("B", 3, "C", 1)} : B gagne ($3 > 1$) $\to$ B reçoit 3 points.
\end{itemize}

Résultat : \lstinline{{"A": 4, "B": 3, "C": 1}}.
\end{corrige}
