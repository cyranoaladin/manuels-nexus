\begin{corrige}{1NSI-TC-EX-008}
On parcourt la liste en retenant le tuple ayant la plus grande température :

\begin{python}
def station_la_plus_chaude(stations):
    # On initialise avec la premiere station
    meilleure = stations[0]
    for station in stations:
        # On compare les temperatures (indice 1)
        if station[1] > meilleure[1]:
            meilleure = station
    # On renvoie uniquement le nom (indice 0)
    return meilleure[0]
\end{python}

La variable \lstinline{meilleure} stocke le tuple complet de la station candidate. À chaque itération, si la température courante dépasse le maximum connu, on met à jour \lstinline{meilleure}. En fin de boucle, \lstinline{meilleure[0]} donne le nom recherché.

Vérification : \lstinline{station_la_plus_chaude([("Paris", 32), ("Lyon", 35), ("Lille", 28)])} renvoie bien \lstinline{"Lyon"}.
\end{corrige}
